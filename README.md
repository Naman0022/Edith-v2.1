# Transliteration
Transliteration refers to the method of mapping from one system of writing to another based on phonetic similarity.  
like  
kya -> क्या ->(eng)-> What  
and badhiya -> बढ़िया ->(eng)-> good  
It is currently in progress...wait fpr version-2.2 :)
